## 解决方案
登录行为识别赛题主要是根据登录行为来判断本次交易是否是危险交易。我们的方案主要是通过刻画id画像、ip画像和dev画像的思路来构造特征，然后分别用xgboost、LR来预测，
取两个模型中风险最大的top200，对这200个进行交集运算。对于某次交易，如果xgboost和LR模型同时预测为危险交易，则判定该次交易为危险交易；否则为正常交易。特征构造
